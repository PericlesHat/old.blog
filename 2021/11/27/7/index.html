人工智能、机器学习和深度学习

人工智能（英语：artificial intelligence，缩写为AI）指由人制造出来的机器所表现出来的智能。

人工智能可以定义为模仿人类与人类思维相关的认知功能的机器或计算机，甚至是算法。人工智能是计算机科学的一个分支，它感知其环境并采取行动，最大限度地提高其成功机会。此外，人工智能能够从过去的经验中学习，做出合理的决策，并快速回应。因此，人工智能研究人员的科学目标是通过构建具有象征意义的推理或推理的计算机程序来理解智能。

目前，人工智能的四个主要组成部分是：

  • 专家系统：作为专家处理正在审查的情况，并产生预期或预期的绩效
  • 启发式问题解决：包括评估小范围的解决方案，并可能涉及一些猜测，以找到接近最佳的解决方案
  • 自然语言处理：在自然语言中实现人机之间的交流
  • 计算机视觉：自动生成识别形状和功能的能力

目前人工智能发展仍处于弱人工智能阶段。AI不具有自主意识，只是帮助人类解决问题的工具。

机器学习（英语：machine learning）是人工智能的一个分支。人工智能的研究历史有着一条从以“推理”为重点，到以“知识”为重点，再到以“学习”为重点的自然、清晰的脉络。显然，机器学习是实现人工智能的一个途径，即以机器学习为手段解决人工智能中的问题。机器学习在近30多年已发展为一门多领域交叉学科，涉及概率论、统计学、逼近论、凸分析、计算复杂性理论等多门学科。机器学习理论主要是设计和分析一些让计算机可以自动“学习”的算法。机器学习算法是一类从数据中自动分析获得规律，并利用规律对未知数据进行预测的算法。因为学习算法中涉及了大量的统计学理论，机器学习与推断统计学联系尤为密切，也被称为统计学习理论。

深度学习（英语：deep learning）是机器学习的分支，是一种以人工神经网络为架构，对资料进行表征学习的学科。深度学习是复杂的机器学习算法，在语音和图像识别方面取得的效果，远远超过先前相关技术。至今已有数种深度学习框架，如深度神经网络、卷积神经网络和深度置信网络和循环神经网络已被应用在计算机视觉、语音识别、自然语言处理、音频识别与生物信息学等领域并获取了极好的效果。

区别于传统的浅层学习，深度学习的不同在于： 强调了模型结构的深度，通常有10多层的隐层节点；明确了特征学习的重要性。也就是说，通过逐层特征变换，将样本在原空间的特征表示变换到一个新特征空间，从而使分类或预测更容易。与人工规则构造特征的方法相比，利用大数据来学习特征，更能够刻画数据丰富的内在信息。

神经网络

人类的大脑本身就是一个超级神经网络。人从出生的那一刻开始，就每时每刻都在接受外界的输入，因此人每时每刻都在学习。我们今天可以快速区分不同人、不同生物、不同物体，可以解决困难的理工问题，可以有复杂的情绪感受，都是大脑十多二十年来学习的成果。

计算机科学家们模仿人大脑神经元的结构设计了神经网络。下图是一个典型的神经元结构。

人工神经网络（Artificial neural networks，ANN），通常简单地称为神经网络（Neural networks， NN），是受构成动物大脑的生物神经网络启发的计算系统。人工神经网络是基于被称为人工神经元（Neuron）的连接单元或节点的集合，它松散地模拟了生物大脑中的神经元。每个连接就像生物大脑中的突触一样，可以向其他神经元传递信号。神经元接收信号，然后对其进行处理，并可向与其相连的神经元发出信号。连接处的 “信号 “是一个实数，而每个神经元的输出是由其输入之和的一些非线性函数计算出来的。这些连接被称为边。神经元和边通常有一个权重（weight），随着学习的进行而调整。权重增加或减少了连接处的信号强度。神经元可能有一个阈值，只有当总体信号越过该阈值时才会发送信号。通常情况下，神经元被聚合成层（layer）。不同的层可以对其输入进行不同的转换。信号从输入层（Input layer）到输出层（Output layer），可能要经过多次穿越各层，中间的各层通常称为隐藏层（Hidden layer）。

我们需要向神经网络输入我们的训练集进行模型（Module）的训练。一般训练集的特征数目就是神经网络输入层的神经元数目（如：训练一个 28*28 的图片数据集，输入层有 784 个神经元）。神经网络通过前向传播（Forward Propagation）学习参数，通过反向传播（Backward Propagation）修正参数。学习好的权重和偏置（bias）等参数组成了我们的模型。我们用模型来测试样本，以验证我们设计的神经网络及其超参数是否合理。验证模型好坏的指标一般是测试精度（Test accuracy），也叫做泛化能力。

如果神经网络在训练时的迭代次数（epoch）过多，有可能产生过拟合（overfitting）现象。过拟合会导致模型的训练精度（Train accuracy）过高，而泛化能力很一般。

如果神经网络训练迭代次数过少，则会产生欠拟合（underfitting）。

卷积神经网络（Convolutional Neural Network, CNN）是一种前馈神经网络，它的人工神经元可以响应一部分覆盖范围内的周围单元，对于大型图像处理有出色表现。

卷积神经网络由一个或多个卷积层和顶端的全连接层（Full- connected layer）组成，同时也包括关联权重和池化层（pooling layer）。这一结构使得卷积神经网络能够利用输入数据的二维结构。与其他深度学习结构相比，卷积神经网络在图像和语音识别方面能够给出更好的结果，使之成为一种颇具吸引力的深度学习结构。

卷积（Convolution）可以使一个图像中的像素产生空间联系（图像中的像素是有序的，因此必须在神经网络中体现出其空间关系，而不能简单地直接输入图像全部像素）。因此卷积神经网络常用来处理图像。卷积神经网络是计算机视觉的核心深度学习结构。

LeNet是最早发布的卷积神经网络之一，因其在计算机视觉任务中的高效性能而受到广泛关注。 这个模型是由 AT&T 贝尔实验室的研究员 Yann LeCun 在1989年提出的（并以其命名），目的是识别图像 [LeCun et al., 1998] 中的手写数字。 当时，Yann LeCun 发表了第一篇通过反向传播成功训练卷积神经网络的研究，这项工作代表了十多年来神经网络研究开发的成果。

YOLO目标检测

YOLO（You Only Look Once）是一种基于深度神经网络的目标检测算法，其最大的特点是运行速度很快，可以用于实时系统。现在YOLO已经发展到v5版本。YOLO具有快速、精确和易于训练的特点，在实时物体检测方面有着悠久而成功的历史。

YOLO可以将图片或视频中的目标对象标记并定位出来。这些对象既可以是YOLO预训练好的模型提供的，也可以是你自己训练特定需求的模型提供的。一个运行默认YOLOv5m模型的目标检测效果如下：

安装PyTorch

PyTorch 是 Python 的第三方包，是一个广泛使用的深度学习框架，集成了多种深度学习模型和算法。

PyTorch 有 CPU 版本和 GPU 版本（以下简称 Cuda 版本），顾名思义就是用中央处理器运行的和用显卡运行的版本。在深度学习训练过程中，GPU 运行速度会远大于 CPU。

你首先需要确定你的电脑的显卡性能，如果是集成显卡或者比较普通的独立显卡，可能没有必要折腾Cuda版本的安装。如果有类似 GTX1070、RTX2060 或以上的显卡，那么推荐安装 Cuda 版本的 PyTorch。注意，仅英伟达（NVIDIA）显卡支持 Cuda 版本，AMD 及其他品牌的显卡不支持。

CPU版本安装过程：

  1. 在命令行（cmd）中使用 conda create -n mypytorch命令来新建一个名为 mypytorch 的 python 环境。这非常重要，因为只有在新环境安装依赖库才不会影响到其他的环境。之后你每次需要使用 PyTorch，需要在编辑器中（PyCharm 或 VSCode）上选择好环境，或者在命令行中使用 conda activate mypytorch 来启用该环境。
  2. 更换清华的 anaconda 镜像：https://mirrors.tuna.tsinghua.edu.cn/help/anaconda/。修改用户目录（C:\Users\xxx）下的 .condarc 文件来更换。
  3. 在 Github 中下载对应 YOLO 代码，并解压到本地。你可以任意选择 YOLOv3 - v5 的代码尝试。这里推荐一个非官方渠道的 YOLOv4 代码仓库，他配套了一个非常完善的 YOLOv4 的视频讲解。

  1. 在 cmd 中使用 cd PATH （PATH 为 YOLO 文件夹路径）进入代码路径，然后使用 pip install -U -r requirements.txt 来安装依赖。注意一定是在新环境（mypytorch）下安装。如果你在前面成功更换了清华源，这一步应该不会卡死。如果卡死需要重来，记得先运行 conda clean --all 来清楚上一次安装失败的残余文件。
